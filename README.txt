HOW TO REQUEST DATA: A javascript file will be set up requiring Seneca to work as the broker. Via node.js, Seneca can be requested to broker a service.
HOW TO RECEIVE DATA: Seneca will provide a listen on a port. When a call is made on that port, Seneca will send the requested data.
